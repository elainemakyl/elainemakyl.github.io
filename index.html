Image Processing for Breast Cancer

The project was done when I was doing the internship in the University at Buffalo.

The aim of this project is to detect tumor regions in a breast in an image retrieved from a photoacoustic imaging system using MATLAB. A series of image processing methods were chained and applied to the images to reduce noises first then to find potential tumor regions.
